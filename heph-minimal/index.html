Designing Adventures

Hephaestus is a freeware computer role-playing game construction kit (often called an "RPG maker"). You can write your own adventures in a simple yet powerful scripting language, and share them with anyone else who has Hephaestus. Hephaestus is written in Java, and therefore runs on every platform.

Hephaestus uses the art library of Lost Dragon. Thanks, DeBray!

Mythology

Hephaestus is the Greek god of the forge, art, and industrialization; he is the god of engineers*.

Hera's bastard son, Zeus cast him out of Olympus and he fell for a day and a night to the isle of Lemnos, leaving him lame.

During the war with the Titans, he created thunderbolts to arm the gods; during the Trojan war, he made impenetrable armor for Æneas; and for his own use he made tripods which would move at his command - the first robots!

Download & Installation

  1. You must have a 1024x768 display with 24-bit or at least 16-bit color depth, and the Java Runtime Environment (JRE), version 1.3 or later. Linux users can find the most current JRE at Blackdown.org.
  2. Download and unzip hephaestus.zip (you can unzip it anywhere).
  3. Now start the game:
    Unix:
    Run ./heph.sh
    Windows:
    Run heph.bat
    MacOS X:
    Double-click on hephaestus.jar, or open a terminal window and run ./heph.sh
    Other:
    If you run Hephaestus on any other platform (BeOS, OS/2, MacOS 9-, etc.), let me know how you did it and how well it worked, and I'll add instructions for your platform.
  4. Select "New Adventure" from the "Game" menu, and pick "Hades.adv", or any other adventure you've downloaded or created. You should see something like the screenshot at the top of this page.
  5. That's it! You're ready to play!

Having Problems?

  • There may still be problems with focus on JDK 1.4, but I think I have corrected the main defect. If not, you will probably have to run Hephaestus with JDK 1.3 until the next version of Hephaestus, which will almost certainly require JDK 1.4. Sun completely revised the focus system in 1.4, and it is not fully backwards-compatible.
  • If you get an error message when you run 'heph.bat' or 'heph.sh' saying something like "class java/lang/Thread not found", or "-jar: unknown switch", or "java: command not found", you don't have Java 1.3 or later installed. See Installation step 1. In particular, a lot of systems (including SuSE Linux, which is what I use) still have Java 1.1 installed, which is about 4 years obsolete; 1.1 is okay for old applets, but it's not advanced enough for Hephaestus.

    You can type the following command at your shell (this works in Unix, Windows, and MacOS X) to see which version of Java you have installed:

    % java -version
  • Display speed may be sensitive to your screen setup. If it's slow at 24-bit, try 16-bit; if it's slow at 16-bit, try 24-bit or 32-bit. It does about 10 frames per second on my K6-2 550MHz w/128MB RAM and a crappy video card; on a serious modern machine, you should never get display lag.
  • If none of these solve your problem, email me at <kamikaze@kuoi.asui.uidaho.edu> or join the hephaestus-user mailing list.
    If you're not having problems, join up anyway, I'm lonely.

Mailing Lists

There are three mailing lists for Hephaestus:

  • Hephaestus Announce, a low-volume moderated list for announcements and news.
  • Hephaestus User, an open list for players of Hephaestus.
  • Hephaestus Design, an open list about designing adventures for Hephaestus.
